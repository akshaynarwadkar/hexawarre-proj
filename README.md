# Hexaware Project

This is a responsive web application built with React, TypeScript, and Tailwind CSS. It features a modern design with animations, a responsive grid layout, and interactive components.

## Prerequisites

- Node.js (v22.13.1 or higher recommended)
- npm

## Getting Started

1.  **Clone the repository** (if you haven't already):
    ```bash
    git clone <repository-url>
    cd hexaware-proj
    ```

2.  **Install dependencies**:
    ```bash
    npm install
    ```

3.  **Run the development server**:
    ```bash
    npm run dev
    ```
    Open [http://localhost:5173](http://localhost:5173) in your browser.

## Building for Production

To create a production build:

```bash
npm run build
```

The output will be in the `dist` folder.

## Features implemented

-   **Responsive Design**: Mobile-first approach, ensuring layouts adapt seamlessly from mobile to desktop.
-   **Animations**: Hover effects on cards and buttons.
-   **Component-Based Architecture**: Modular components for Header, Hero, FilterBar, ProjectGrid, etc.
-   **Images**: Integrated assets for a rich visual experience.
